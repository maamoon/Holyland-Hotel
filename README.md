# Holyland Hotel Amman

Official landing page for Holyland Hotel — 3-star hotel on Al Jama'a Street, Amman, Jordan.

## Live Site

[View on GitHub Pages](https://YOUR_USERNAME.github.io/holyland-hotel-site/)

## Features

- Contact info (phone, email, WhatsApp)
- Social links (Instagram, Facebook, Google)
- Review links (Google, Tripadvisor, Facebook)
- Photo gallery
- Book direct link to [holyland-hotel.com](https://www.holyland-hotel.com/en/)

## Local Development

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Deploy

This site is configured for GitHub Pages. Push to the `main` branch to deploy.
